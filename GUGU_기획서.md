# GUGU. 서비스 기획서
> 인플루언서 공동구매(공구) 플랫폼
> 업데이트: 2026-04-09

---

## 1. 서비스 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | GUGU. |
| 핵심 개념 | 인플루언서가 직접 공구를 열고, 팔로워(소비자)가 참여해 구매하는 플랫폼 |
| 주요 사용자 | 소비자(팔로워), 인플루언서, 관리자(어드민) |
| 결제 방식 | 토스페이먼츠 v1 (신용카드 / 카카오페이 / 네이버페이 / 계좌이체) |
| 배포 | Vercel (자동 배포) |

---

## 2. 역할(Role) 구조

```
consumer   — 기본 가입자. 공구 탐색 및 구매
influencer — 공구 등록자. 관리자 승인 후 활성화
admin      — 플랫폼 전체 관리 (주문·인플루언서·공구)
```

---

## 3. 소비자(팔로워) 워크플로우

```
① 진입
   메인홈 (index)
     ├─ 카테고리 탐색 (category)
     ├─ 검색 (search)
     └─ 인플루언서 프로필 (influencer-profile)

② 상품 확인
   상품 상세 (product-detail)
     - 공구 가격 / 남은 기간 / 참여 현황(진행률)
     - 인플루언서 정보 / 팔로우 버튼
     - 관련 공구 추천

③ 구매
   로그인 (login) ← 비로그인 시 자동 이동
     ↓
   결제 (checkout)
     - 배송지 입력 (이름·연락처·주소)
     - 결제 수단 선택
     - 토스페이먼츠 결제창 실행
     ↓
   결제 완료 (order-complete)
     - 서버에서 결제 최종 확인 후 주문 저장

④ 사후 관리
   마이페이지 (mypage)
     - 주문 내역 / 배송 단계 표시
     - 주문 취소 (마감 24시간 전까지)

   알림 (notifications)
     - 실시간 알림: 배송 시작·완료, 팔로우 인플루언서 새 공구 오픈
```

---

## 4. 인플루언서 워크플로우

```
① 신청
   인플루언서 랜딩 (influencer-landing)
     ↓
   신청 폼 (influencer-apply)
     - 채널명 / SNS URL / 팔로워 수 / 카테고리 / 소개글
     - 회원가입 동시 진행
     ↓
   관리자 검토 → 승인 시 role = influencer 전환
   (승인/거절 알림 수신)

② 공구 운영
   로그인 (login)
     ↓
   인플루언서 대시보드 (influencer-dashboard)
     - 이번달 예상 수익 / 진행 중 공구 수 / 누적 참여자
     - 최근 주문 목록 / 공구별 매출 현황
     ↓
   공구 등록 (open-gugu)
     - 상품명 / 카테고리 / 정가·공구가 / 기간 / 목표 인원 / 설명
     ↓
   소비자에게 상품 노출 → 알림 발송

③ 정산
   (현재 개발 중)
   공구 마감 후 판매 금액의 95% 정산 예정
```

---

## 5. 관리자 워크플로우

```
admin.html 접근 (admin role 계정만)

① 대시보드
   - 오늘 매출 / 이월 매출 / 배송 대기 / 신규 신청 현황

② 주문 관리
   - 전체 주문 조회 (상태 필터: 결제완료·준비중·배송중·배송완료·취소)
   - 상태 변경 / 송장번호·택배사 입력
   - 상태 변경 시 소비자에게 자동 알림

③ 인플루언서 신청 관리
   - 신청 목록 (대기·승인·거절)
   - 승인 시 → 계정 role을 influencer로 변경 + 알림
   - 거절 시 → 사유 포함 알림 발송

④ 공구 목록
   - 전체 공구 현황 (진행률 / 가격 / 상태)
```

## 6. 현재 구현 현황

### ✅ 완료
- 메인홈 · 카테고리 · 검색 · 상품 상세 (Supabase 실시간 연동)
- 회원가입 / 로그인 (이메일 + 카카오 + Google)
- 결제 전체 흐름 (checkout → 토스페이먼츠 → order-complete → DB 저장)
- 주문 내역 · 배송 단계 표시 (마이페이지)
- 실시간 알림 (Supabase Realtime)
- 인플루언서 신청 폼 · 대시보드 · 공구 등록
- 관리자 패널 (주문관리 · 인플루언서 승인 · 공구 목록)
- 팔로우 API

### 🔴 미완성 — 우선순위 높음

| 항목 | 내용 |
|---|---|
| 마이페이지 송장번호 표시 | 배송 시작 후 송장번호·택배사 UI 없음 |
| 주문 취소 버튼 | cancel API는 완성, mypage UI 연결 필요 |
| 인플루언서 공구 수정/삭제 | 등록만 가능, 수정·삭제 UI 없음 |
| 팔로우 버튼 연결 | 인플루언서 프로필 팔로우 버튼 API 미연결 |

### 🟡 미완성 — 우선순위 보통

| 항목 | 내용 |
|---|---|
| 정산 화면 | 인플루언서 정산 내역 페이지 없음 |
| 리뷰 기능 | 배송 완료 후 리뷰 작성 화면 없음 |
| 찜 기능 | UI는 있지만 DB 미연결 |
| 공구 자동 마감 | end_date 지나면 status 자동 변경 처리 없음 |

### 🟢 추가 검토 필요

| 항목 | 내용 |
|---|---|
| SNS 공유 | 공구 링크 카카오·인스타 공유 기능 |
| 인플루언서 승인 후 온보딩 | 승인 완료 후 대시보드 안내 흐름 |
| 고객센터 | help.html 존재하지만 실제 문의 기능 없음 |
| 네이버 로그인 | 카카오·Google은 완료, 네이버 미구현 |
